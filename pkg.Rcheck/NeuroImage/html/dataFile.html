dataFile {NeuroImage}R Documentation

access the name of the data file associated with a "BrainHeader" object

Description

acesses the name of the file stored in a "BrainHeader" instance.

Usage

dataFile(x)

Arguments

x object deriving from class "BrainHeader"

Value

a character string returning the filename of the image data.
