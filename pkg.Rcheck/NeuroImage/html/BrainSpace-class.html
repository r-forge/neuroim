BrainSpace-class {NeuroImage}R Documentation

Class "BrainSpace"

Description

BrainSpace represents the topology of a dataset

Objects from the Class

Objects can be created by calls of the form BrainSpace(Dim, dimension, spacing, origin) .

Slots

Dim:
Object of class "integer" – an integer array specifiying the spatial dimensions
origin:
Object of class "numeric" – an array representing the absolute location of the origin (coordinate at voxel [0, 0, 0])
spacing:
Object of class "numeric" – an array representing the grid spacing for each dimension

orientation:
Object of class "character" – character array describing axes orientation, e.g., "L" for "left-right", "P" for "posterior-anterior", "I" for "inferior-superior". This slot is currently not used and may contain invalid information.
trans:
Object of class "matrix" – This is a transformation matrix used for computing the mapping from index coordinates to real world oordiantes. Should be a 4X4 matrix for volumetrix data.
invTrans:
Object of class "matrix" – This is the inverse transformation (real world to index) and is computed internally and therefore should not be changed externally
reptime:
Object of class "numeric" – The delay between volume acquisitions in seconds

Methods

bounds
signature(x = "BrainSpace"): – returns the real world bounds as a matrix of (min, max) pairs
invTrans
signature(x = "BrainSpace"): – returns the dimension of the data axes for the space
numdim
signature(x = "BrainSpace"): – returns the number of dimensions in space
orientation
signature(x = "BrainSpace"): – returns the (currently unsupported) orientation code
origin
signature(x = "BrainSpace"): – returns the coordinate origin
spacing
signature(x = "BrainSpace"): – returns the voxel spacing
trans<-
signature(x = "BrainSpace", value = "matrix"): – returns the index-to-word transformation matrix
trans
signature(x = "BrainSpace"): – returns the inverse transformation matrix
reptime
signature(x = "BrainSpace"): – returns the repetition time
show
signature(x = "BrainSpace"): – print information about BrainSpace object
