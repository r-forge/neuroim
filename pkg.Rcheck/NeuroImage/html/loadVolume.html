loadVolume {NeuroImage}R Documentation

loads an image volume from file

Description

loads an image volume from file and creates object of class "BrainVolume"

Usage

loadVolume(file, volNum = 1)

Arguments

file the name of the image file to load
volNum the volume index if the file is 4 dimensional (starts at 1)

Value

an object of class "BrainVolume"
