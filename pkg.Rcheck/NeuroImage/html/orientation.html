orientation {NeuroImage}R Documentation

function returns the orientation of a brain image

Description

generic function that returns a character string coding the orientation of a brain image

Usage

orientation(x)

Arguments

x an object deriving from class "BrainSpace" or "BrainData"

Details

The orientation string encodes the anatomical direction of each of the (spatial) image axes. Thus, a standard "neurologically" oriented image volume has the orientatrion string "LPI", whcih stands for "Left, Posterior, Inferior". Currently this function is used only for information.

Value

character string representing image orientation
