readData {NeuroImage}R Documentation

reads image data into an appropriate "BrainData" object

Description

function to load image data into appropriate data structure, e.g. an object derived from the "BrainData" class

Usage

readData(x, indices)

Arguments

x an object derived from the "BrainFile" class
indices the 1D indices of the image cells to load

Value

a "BrainData" object appropriate for the type of data file being read
