BrainData-class {NeuroImage}R Documentation

Virtual Class "BrainData"

Description

The BrainData class is a class contained by all actual classes in the Neuroimage package. It is a “virtual” class.

Objects from the Class

A virtual Class: No objects may be created from it.

Slots

Common to all BrainData objects in the package:

space:
Object of class "BrainSpace" – a description of the data "space", e.g. voxel resolution and iamge dimensions.

Methods

dim
signature(x = "BrainData"): extract data dimensions dim.
space
signature(x = "BrainData"): extract “space” slot.
numdim
signature(x = "BrainData"): return number of dimensions in space
spacing
signature(x = "BrainData"): returns cell spacing for image data
