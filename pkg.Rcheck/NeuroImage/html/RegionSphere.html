RegionSphere {NeuroImage}R Documentation

creation function for Spheric BrainRegion3D

Description

creates a new BrainRegion3D instance with a spheric geometry.

Usage

        RegionSphere(bvol, centroid, radius)

Arguments

bvol the BrainVolume that on which the geometry of the region is based
centroid the center coordinates (in grid space) of the region
radius the radius of the sphere of the constructed region
