function to create a an instance of the "NIFTIHeader" class

Description

function to creatte a n instance of the "NIFTIHeader" class

Usage

NIFTIHeader(headerFile, dataFile, extraInfo, fileType = "NIFTI", versionNum = "1", encoding = extraInfo$encoding, endian = .Platform$endian, dataOffset = 348, dataType = NULL)

Arguments

headerFile the path to the header file
dataFile the path to the data file (may be same as above)
extraInfo list containing auxiallary header information
fileType the file format
versionNum the version of the file format
encoding the encoding type ("binary", or "gz")
endian the endianess of the stored image data
dataOffset the byte offset in the image file where the first value may be found
dataType the stored dataType

Value

an instance of the "NIFTIHeader" class
