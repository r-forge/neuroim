encoding {NeuroImage}R Documentation

retrieves the encoding method used for image data

Description

accessor for the way in which the image data is encoded on disk (e.g. "raw", "gzip", etc.")

Usage

encoding(x)

Arguments

x an object deriving from class "BrainHeader"

Value

a character string representing the encoding type
