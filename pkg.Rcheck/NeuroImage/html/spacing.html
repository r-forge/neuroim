spacing {NeuroImage}R Documentation

returns cell dimensions for a voxel or pixel unit

Description

generic function to access the voxel or pixel dimension for a brain image

Usage

spacing(x)

Arguments

x object derived from "BrainSpace" or "BrainData" classes

Value

a numeric vector of the cell sizes for each image axis

Note

this function does not deal with non-spatial dimensions, such as time. Thus, a 4-D fMRI image return a vector of length 3, one for each spatail dimension.
