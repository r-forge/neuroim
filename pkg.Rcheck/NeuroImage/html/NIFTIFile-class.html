NIFTIFile-class {NeuroImage}R Documentation

Class "NIFTIFile"

Description

a class that handles input/output of NIFTI formatted image data

Objects from the Class

Objects can be created by calls of the form new("NIFTIFile", ...).

Slots

fileType:
Object of class "character" : the file type
path:
Object of class "character" : the file path
open:
Object of class "character" : whether to open file for reading "r" or writing "w"

Extends

Class "BrainFile", directly.
