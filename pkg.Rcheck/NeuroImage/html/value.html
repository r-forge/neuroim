value {NeuroImage}R Documentation

a function to access the value given a real world coordinate

Description

this function takes real world coordinate locations and returns an interpolated image value.

Usage

value(object, x, y, ...)

Arguments

object an object derived from class "BrainData"
x the x real world coordinate
y the y real worold coordinate
... world coordinates for additional dimensions

Details

coordinates supplied to the function will first be multiplied by the inverse transformation matrix associated with the "BrainSpace" object, yileding a set of numeric grid positions. An image value will be computed via linear interpolation.

Value

interploated value at real world coordinates.
