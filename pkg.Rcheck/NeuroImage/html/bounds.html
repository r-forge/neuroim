bounds {NeuroImage}R Documentation

get the real world bounds of an image's data space

Description

generic method for accessing the real world coordinates bounds of a BrainSpace object.

Usage

bounds(x)

Arguments

x an object from which to retrieve the bounds

Examples


space <- BrainSpace(c(256,256))
b <- bounds(space)
