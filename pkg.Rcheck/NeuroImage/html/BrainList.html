BrainList {NeuroImage}R Documentation

creates a new "BrainList" object

Description

function to create a new object of class "BrainList"

Usage

BrainList(volList, labels = NULL, files = NULL)

Arguments

volList the list of BrainData objects
labels the names of the objects, optional
files the file names of the images, optional

Details

This class is used for storing related collections of brain images. It is simply a class extending list and BrainData. For instance, one may represent a set of MRIs across a group of subjects, each having the same geometry, as a "BrainList". Unlike, BrainVector instances, which are stored as arrays and suitable for representing multi-dimensional images such as FMRI time-series, BrainList objects are better suited for loose, possibly unordered collections of related data sets. Thus, "BrainList" objects are closely related to "bucket" data sets in AFNI.

Value

an object of class "BrainList"
