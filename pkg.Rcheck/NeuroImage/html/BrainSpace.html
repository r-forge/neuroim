BrainSpace {NeuroImage}R Documentation

creates a new "BrainSpace" object

Description

function to create a new "BrainSpace" object from scratch

Usage

BrainSpace(Dim, origin = NULL, spacing = NULL, orientation = NULL, trans
= NULL, reptime=1)

Arguments

Dim an integer vector supplying the dimensions of the image grid
origin a numeric vector with length equal to Dim represting image origin in mm
spacing the cell (pixel or voxel) size for each dimension in mm
orientation a character vector representing anatomical orientation of the grid axes
trans a transformation matrix used to convert grid positions to real world coordinates
reptime the delay in seconds between successive volume acquisitions

Value

an object of class "BrainSpace"
