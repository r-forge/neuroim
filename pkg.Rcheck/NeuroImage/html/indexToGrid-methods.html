indexToGrid-methods {NeuroImage}R Documentation

converts a 1-dimensional index vector to a matrix of coordinates

Description

function to convert from 1-dimensional index coordinates to an n-dimensional coordinate representation.

Methods

x = "BrainSlice", idx = "index"
converts a 1-dimensional index vector to a two-column matrix of coordinates
x = "BrainSpace", idx = "index"
converts a 1-dimensional index vector to a n-column matrix of coordinates
x = "BrainVolume", idx = "index"
converts a 1-dimensional index vector to a three-column matrix of coordinates
