trans {NeuroImage}R Documentation

function to access the forward transformation associated with a brain image

Description

this function access the transformation matrix attached to a "BrainSpace" object

Usage

trans(x)

Arguments

x an object derived from "BrainData" or "BrainSpace" classes

Details

transformation matrices are 4X4 affine matrices for 3D images and 3X3 matrices for 2D images

Value

the transformation matrix
