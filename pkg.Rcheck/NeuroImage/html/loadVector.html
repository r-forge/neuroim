loadVector {NeuroImage}R Documentation

convenience function to create a "BrainVector" instance from a specified file

Description

convenience function to create a "BrainVector" instance from a specified file

Usage

loadVector(filename, volRange = NULL, mask=NULL)

Arguments

filename name of the file to load
volRange the volume indices to load
mask a mask, which can be an array or BrainData instance with appropriate dimensions, indicating which voxels to load. If provided function returns a SparseBrainVector

Value

an of object of class BrainVector
