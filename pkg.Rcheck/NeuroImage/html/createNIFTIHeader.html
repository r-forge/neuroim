createNIFTIHeader {NeuroImage}R Documentation

a function to create a NIFTIHeader object

Description

this function may be used to create a NIFTIHeader object given an existing brain data object

Usage

createNIFTIHeader(data, fileName, dataType = "FLOAT")

Arguments

data an object deriving from the class BrainData
fileName the file name for the header (need not exist)
dataType the storage type of the data, e.g. "SHORT"

Details

This argument is useful if one wants to create a NIFTIHeader object on the basis of an existing BrainData object

Value

an object of class NIFTIHeader
