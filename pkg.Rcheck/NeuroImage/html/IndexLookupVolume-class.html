IndexLookupVolume-class {NeuroImage}R Documentation

A class used to map from 1d indices to lookup table indices

Description

An IndexLookupVolume maps from 1-d volume indices to an ordered set of indices that map to indices ina lookup table. This class is currently used in the implementation of SparseBrainVector

Objects from the Class

Objects can be created by calls of the form new("IndexLookupVolume", space, indices).

Slots

space:
Object of class "BrainSpace"
indices:
Object of class "integer"
map:
