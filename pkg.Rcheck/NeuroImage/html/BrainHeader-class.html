Class "BrainHeader"

Description

A virtual class representing basic information contained in brain image headers

Objects from the Class

A virtual Class: No objects may be created from it.

Object of class "character" : the name of the header file
dataFile:
Object of class "character" : the name of the data file
fileType:
Object of class "character" : the file format of this header
versionNum:
Object of class "character" : the version number of the file format
encoding:
Object of class "character" : the encoding type, which may be "binary" or "gz"
endian:
Object of class "character" : the endianess of the stored data
dataOffset:
Object of class "numeric" : the offset in the data file where the first image value may be found
dataType:
Object of class "character" : the data type of the stored image data
extraInfo:
Object of class "list" : auxilliary or format-specific information
