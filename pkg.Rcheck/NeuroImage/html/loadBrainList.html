loadBrainList {NeuroImage}R Documentation

a function to create a "BrainList" object from a simple data.frame descriptor

Description

a convenience function to create a "BrainList" object from a text file description

Usage

loadBrainList(tableSpec, path = NULL)

Arguments

tableSpec a two-column data.frame containing the labels and image file names
path optional path to image files

Details

the supplied data.frame must have two columns named "label" (for the image identifiers) and "file" for the image file names. These are then used as key-value paris in the constructed BrainList object. It is assumed that each opf the imege files are three-dimensional and this can be loaded with the loadVolume function.

Value

an object of class BrainList
