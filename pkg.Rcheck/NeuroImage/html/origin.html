origin {NeuroImage}R Documentation

retrieves the image origin

Description

the image is the real world coordinate location at the first grid location of the data array (e.g. voxel [1,1,1] )

Usage

origin(x)

Arguments

x object deriving from class "BrainSpace" or "BrainData"

Value

a numeric vector indicating the location in real world coordinates.
