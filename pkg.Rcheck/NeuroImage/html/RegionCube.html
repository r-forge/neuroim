RegionCube {NeuroImage}R Documentation

creates a BrainRegion3D instance that is a cube

Description

creation fucntion for a BrainRegion3D class that is a cube.

Usage

 RegionCube(bvol, centroid, surround, mask)

Arguments

bvol the BrainVolume that on which the geometry of the region is based
centroid the center coordinates (in grid space) of the region
surround the number of surrounding voxels on each side of the center voxel
mask a mask supplied as a 3d array the same shape as bvol that will be used to exlucde voxels in the cube that fall outside the mask
