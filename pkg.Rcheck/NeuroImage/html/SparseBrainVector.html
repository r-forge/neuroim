SparseBrainVector {NeuroImage}R Documentation

a function to create a new SparseBrainVector instance

Description

A convenience creation function for the SparseBrainVector class. This function can construct a 4-dimensional sparse data structure from a matrix of values and a mask or 1d index vector.

Usage

        SparseBrainVector(dat, space, mask = NULL, indices = NULL)

Arguments

dat a matrix of values where the number of rows or columns must equal the number of elements in the mask or ndices arguments.
space a BrainSpace instance that encodes the 3d geometry of the to-be-created structure
mask an mask which must have the same dimensions as the space argument.
indices an alternative to mask (either one or the other must be supplied) which is simple a 1d vector of indices that map to the 3d coordinates.

Value

an instance of the SparseBrainVector class
